# RP2040 Based Custom Devboard

This is a rp2040 based devboard based on the pico dimensions with complete battery management and usb c connectivity.
![Board Image](/assets/image.png)

## Features

- RP2040 Microcontroller
- USB-C Connectivity
- Battery Management System
- Multiple GPIO Pins
- Onboard LEDs
- Onboard Buttons
- Compact Design
